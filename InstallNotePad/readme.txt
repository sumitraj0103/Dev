================
InstallNotePad.exe is for silently installl the Notepad with ProgressBar.

Steps.

1. Run the InstallNotePad.exe as Administartion 
2. Software folder where the Notepad installer avaiable.
3. Make sure the Software folder available at the InstallNotePad.exe file.

WFP Project.

1. Open the Project in visual Studio
2. Compile the Project
3. Go to \bin\debug and find the executable


Test:
1. Go to Control Panel and check the Notepad installed
2. if it is installed. Uninstall 
3. Run the Package
4. After successful run. Go to Control panel and check the software has been installed or not
